UI constraints

The existing 300 px panel is a useful constraint. Four channels need to be legible without turning the extension into a DAW or covering a meaningful portion of the video.

Keep stable

Panel title, global On/Off, file or ZIP selector, floating speaker button, and YouTube-driven transport.

Add

A channel name derived from its ZIP entry basename, volume value, Mute, and Solo, plus clear effective state.

Avoid

Waveforms, pan, metering, offsets, effects, master transport, and controls that imply unsupported DAW behavior.

Three options

Each panel shows the same four-stem transcription mix: bass soloed at 100%, drums and other retained at lower levels, and vocals muted. The difference is how directly the full state can be inspected and changed.

Recommendation

Start with option A. It aligns with the issue’s simple-mixer scope, keeps all state inspectable, and maps directly to an ID-keyed list of track controls.

Ship first

Compact rows, no presets yet.

Use a 300 px panel, 24 px M/S buttons with title tooltips, a horizontal range, visible percentage, and full stem name in a tooltip. Do not add a mixer header or track count. Preserve one global On/Off switch.

Interaction rules

  1. Mute is independent per track.
  2. Any active Solo makes all non-solo tracks effectively muted.
  3. Multiple tracks may be soloed together.
  4. Muted and soloed state remains visibly distinct.
  5. Global Off restores normal YouTube audio without changing the saved mix.
  6. Display names remove the ZIP entry's path and final extension, then capitalize the first character.

Resolved defaults

These conservative choices keep the first version aligned with the issue and preserve the current single-audio experience.

YouTube stays outside

Original YouTube audio remains muted while replacement is On. It is not represented as another mixer channel.

Stem gain starts at 100%

Every imported stem begins at unity gain. YouTube’s volume remains the shared output level, so stored track gains describe only the mix balance.

Single files stay simple

An ordinary non-ZIP audio file retains the existing standalone Volume row. Mute duplicates global Off and Solo has no effect when there is only one track.
